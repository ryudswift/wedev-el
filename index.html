GLOW & PULSE

A classic tech effect. This combines a neon-like `text-shadow` glow with a subtle `transform: scale` pulse, all driven by CSS animations.

Glow & Pulse Effect

This effect uses two CSS `@keyframes` animations applied to the same element. One handles the `text-shadow` for the glow, and the other handles `transform: scale` for the pulse.

HTML

<h1 class="animated-title">YOUR TEXT HERE</h1>

CSS


:root {
    --glow-color: hsl(217, 100%, 65%);
    --text-color: hsl(217, 100%, 85%);
}

.animated-title {
    color: var(--text-color);
    font-family: 'Exo 2', sans-serif; /* Recommended font */
    font-size: 2rem; /* Adjust as needed */
    animation: glow 2s ease-in-out infinite, pulse 2s ease-in-out infinite;
}

@keyframes glow {
    0%, 100% {
        text-shadow: 0 0 10px var(--glow-color), 0 0 20px var(--glow-color);
    }
    50% {
        text-shadow: 0 0 20px var(--glow-color), 0 0 40px var(--glow-color);
    }
}

@keyframes pulse {
    0%, 100% {
        transform: scale(1);
    }
    50% {
        transform: scale(1.05);
    }
}
